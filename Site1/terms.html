Recursos do Titular dos Dados Pessoais
O titular dos dados pessoais ou seu representante tem o direito de enviar ao Operador um pedido de Processamento ou retirar o Consentimento: por escrito para o endereço especificado; na forma de um documento eletrônico para um endereço de e-mail.
A solicitação deve conter informações que permitam identificar o Titular dos Dados Pessoais, incluindo os dados do seu número de telefone especificados ao fazer um Pedido ou durante o Registro.
A Operadora compromete-se a apreciar o pedido e a enviar-lhe resposta ou recusa fundamentada no prazo de 2 (dois) dias úteis a contar da data de receção do pedido.
,Obrigações do Operador
O Operador é obrigado: a pedido do Titular dos Dados Pessoais, a fornecer informações sobre o Tratamento ou recusa justificada; tomar as medidas necessárias e suficientes para assegurar o cumprimento das obrigações. a pedido do Titular dos Dados Pessoais, esclarecer os Dados Pessoais tratados, bloquear ou eliminar se estiverem incompletos, desatualizados, imprecisos, obtidos ilegalmente ou desnecessários para a finalidade declarada do Tratamento;
assegurar a legalidade do Processamento. Caso seja impossível assegurar a legalidade do Tratamento, o Operador, num prazo não superior a 10 (dez) dias úteis a contar da data de deteção do Tratamento ilegal, obriga-se a destruir os Dados Pessoais ou assegurar a sua destruição;
interromper o Processamento e destruir os Dados Pessoais (se seu armazenamento não for mais necessário para os fins do Processamento) dentro de um período não superior a 30 (trinta) dias a partir da data de recebimento da retirada do Consentimento do Titular dos Dados Pessoais, a menos que o Processamento pode ser continuado.
O Operador implementa as medidas legais, organizacionais e técnicas necessárias e suficientes para garantir a proteção dos Dados Pessoais.
-Medidas legais: elaboração pelo Operador de documentos locais, incluindo a Política e sua colocação no Sistema; recusa de quaisquer métodos de Processamento que não correspondam aos fins predeterminados pelo Operador.
Medidas organizativas: nomeação de um responsável pela organização do Tratamento;
limitar a composição dos funcionários da Operadora que tenham acesso aos Dados Pessoais e organizar um sistema de autorização para acesso aos mesmos;
instruir os funcionários do Operador que realizam o Processamento, inclusive com os requisitos para sua proteção, com os atos locais do Operador que regulam o procedimento de trabalho e proteção de Dados Pessoais; realizar revisões periódicas para controlar internamente a conformidade do Processamento com os requisitos estabelecidos.
Medidas técnicas: prevenção de acesso não autorizado à base de dados na qual os Dados Pessoais são armazenados; backup e restauração de dados pessoais, operacionalidade de hardware e software, ferramentas de segurança da informação em sistemas de informação de dados pessoais; outras medidas de segurança necessárias.
, Privacidade e Proteção da Informação
Ao efetuar Pedidos ou registrar-se no Site, o Comprador preenche os formulários, fornecendo os seguintes dados: Sobrenome, Nome, Nome do meio, dados de contato (telefone, e-mail), Endereço de entrega do Pedido.
O Comprador é responsável pela veracidade das informações fornecidas ao fazer o Pedido.
O Vendedor não é responsável pelo conteúdo e precisão das informações fornecidas pelo Comprador ao fazer um Pedido.
Ao fornecer seus dados pessoais ao Vendedor, o Comprador concorda com seu processamento (até a retirada pelo Comprador de seu consentimento para o processamento de seus dados pessoais), tanto com o uso de ferramentas de automação quanto sem o uso de ferramentas de automação, em particular, a coleta, armazenamento, transferência para terceiros e o uso de informações pelo Vendedor para cumprir as obrigações para com o Comprador de acordo com este Contrato, bem como para promover novos bens e serviços pelo Vendedor e para outros fins especificados neste Contrato. Se o Comprador não quiser que seus dados pessoais sejam processados, ele deverá entrar em contato com o Atendimento ao Cliente do Vendedor usando o e-mail ou número de telefone indicado no Site. Nesse caso, todas as informações recebidas do Comprador (incluindo o endereço de e-mail e a senha) são excluídas da base de clientes do Vendedor.
O processamento de dados pessoais nesta oferta significa: a coleta dos dados acima, sua sistematização, acumulação, armazenamento, esclarecimento (atualização, alteração), uso, distribuição (incluindo transferência internacional), despersonalização, bloqueio, destruição.
O Vendedor é obrigado a garantir a segurança, a segurança dos dados e a confidencialidade das informações recebidas do Comprador de acordo com os requisitos da legislação sobre proteção de dados pessoais.
O Vendedor processa os dados pessoais do Comprador para os seguintes fins:
para registro/autorização do Comprador no Site;
processar pedidos e cumprir suas obrigações para com o Comprador sob um contrato de venda a varejo de mercadorias;
avaliar e analisar o funcionamento do Site;
realizar atividades de promoção de bens e serviços;
para informação informar o cliente sobre promoções, descontos e ofertas especiais por meio de correspondências eletrônicas e SMS;
para entrar em contato com o Comprador.
O Vendedor tem o direito de enviar mensagens de natureza publicitária e informativa ao Comprador. Se o Comprador não desejar receber tais mensagens do Vendedor, ele deverá alterar as configurações apropriadas na Conta Pessoal.
O comprador tem o direito de solicitar ao vendedor informações completas sobre seus dados pessoais, seu processamento e uso.
O Vendedor compromete-se a não divulgar as informações recebidas do Comprador. Não é considerado uma violação que o Vendedor forneça informações a agentes e terceiros agindo com base em um acordo com o Vendedor para cumprir as obrigações para com o Comprador.
Não é considerado uma violação das obrigações divulgar informações de acordo com os requisitos legais razoáveis ​​e aplicáveis.
O Vendedor tem o direito de transferir os dados pessoais do Comprador para as contrapartes do Vendedor (Serviços de Correio) para entregar os Bens encomendados ao Comprador.
O Vendedor tem o direito de usar a tecnologia de "cookies" ao fornecer ao Comprador acesso ao Site. Os cookies não contêm informações confidenciais e não são transferidos para terceiros.
Quando o Comprador acessa o Site, o Vendedor recebe informações sobre o endereço IP do Comprador. O Vendedor compromete-se a não utilizar esta informação para identificar o Comprador.
O Vendedor não é responsável pelas informações divulgadas pelo Comprador através do Site de forma pública.
