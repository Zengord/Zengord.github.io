PARA O TRATAMENTO DOS DADOS PESSOAIS DO CLIENTE
Um indivíduo (doravante denominado Titular dos Dados Pessoais), ao enviar seus dados pessoais por meio de um formulário no site ou em um aplicativo móvel (doravante denominado Sistema), dá seu consentimento ao Operador para seu processamento (doravante denominado como o Consentimento) nas condições estabelecidas abaixo:
O titular dos dados pessoais confirma que é um indivíduo capaz e transfere os seus dados pessoais ao Operador livremente, sem coação, na sua vontade e no seu interesse.
LISTA DE DADOS PESSOAIS, para cujo tratamento é dado o consentimento: dados pessoais: apelido, nome, número de telefone, endereço de e-mail, endereço de entrega, dados do passaporte no envio de mercadorias ou no recebimento de mercadorias pré-pagas.
Informações técnicas: endereço IP, tipo de sistema operacional, tipo de navegador, localização geográfica, provedor de serviços de Internet; informações sobre o comportamento do Titular dos Dados Pessoais no Sistema, informações obtidas automaticamente ao acessar o Sistema, incluindo o uso de cookies, dados sobre o pagamento online: hora, método e valor do pagamento.
FUNDAMENTOS LEGAIS para o tratamento de dados pessoais são: Leis, Política no domínio do tratamento de dados pessoais; aceitação da oferta lançada no Sistema (Contrato de Usuário, Contrato de Compra e Venda de Mercadorias, Contrato de Prestação de Serviços).
OBJETIVO DO PROCESSAMENTO de dados pessoais é:
disponibilização de acesso ao Sistema, inclusive para celebração de contratos com base em ofertas;
identificação do Titular dos dados pessoais, inclusive no acesso ao Perfil;
processamento de Pedidos;
cumprimento das obrigações decorrentes da Oferta;
entrega de mercadorias e transferência de mercadorias para entrega por empresa de transporte;
estabelecer e manter a comunicação com o Titular dos Dados Pessoais, incluindo o envio de: respostas a solicitações; mensagens informativas para o número de telefone e/ou endereço de e-mail: no âmbito da Oferta (estado da encomenda, cheque, etc.), sobre códigos promocionais para as próximas compras, e outras informações de conteúdos publicitários e noticiosos; informações sobre o funcionamento do Sistema.
Garantir a operacionalidade e segurança do Sistema (para prevenir casos de fraude e outros abusos, bem como investigar tais casos);
Solução de problemas do sistema;
Realização de pesquisas estatísticas e mercadológicas de demanda do consumidor, bem como para direcionamento (identificação do público-alvo) de publicidade colocada com base em Dados Pessoais Anônimos;
resolução de disputas;
coleta de feedback;
atendimento dos requisitos estabelecidos.
LISTA DE AÇÕES com dados pessoais: coleta, registro, sistematização, acumulação, armazenamento, esclarecimento (atualização, alteração), cópia eletrônica, extração, uso, transferência (distribuição, disponibilização, acesso), despersonalização, bloqueio, exclusão e destruição.
O consentimento é dado ao processamento de dados pessoais tanto com o uso de ferramentas de automação quanto sem o uso de tais ferramentas.
O titular dos dados pessoais concorda que o Operador tem o direito de transferir dados pessoais para terceiros, incluindo uma empresa de transporte, um serviço de correio para entrega e outras pessoas para o processamento de dados pessoais em nome do Operador para cumprir obrigações e atingir as metas especificadas no Consentimento. Neste caso, o Operador tem o direito de divulgar dados pessoais a essas pessoas na medida do necessário.
CONSENTIMENTO PARA RECEBER LISTAS DE PUBLICIDADE E INFORMAÇÕES. O titular dos dados pessoais, ao registar-se no Sistema, ao efetuar uma Encomenda, compromete-se a receber correspondências de natureza publicitária e informativa do Operador.
O SUJEITO DOS DADOS PESSOAIS TEM O DIREITO DE DESPERTAR as correspondências promocionais e informativas a qualquer momento clicando no link localizado no final de cada mensagem de e-mail de marketing ou no Perfil, desmarcando as notificações e salvando as novas configurações.
O CONSENTIMENTO É VÁLIDO desde o momento em que é concedido e até a assinatura do respectivo contrato ou retirada do Consentimento.
O TITULAR DOS DADOS PESSOAIS TEM DIREITO DE RETIRAR O CONSENTIMENTO mediante o envio de um pedido ao Operador: sob a forma de documento eletrónico para o endereço de correio eletrónico;
O aplicativo deve conter informações que permitam identificar o Titular dos Dados Pessoais, incluindo os dados do seu número de telefone especificados ao fazer um pedido ou registrar-se no Sistema.
