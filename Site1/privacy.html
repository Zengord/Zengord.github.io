Coleta de Dados Pessoais
O Operador recolhe os Dados Pessoais das seguintes formas: O titular dos dados pessoais fornece voluntariamente os dados: ao preencher os formulários do Sistema, utilizando meios de comunicação, nomeadamente por telefone, e-mail, etc.; coleta automática de dados sobre o usuário usando tecnologias, protocolos da web, cookies, tags da web que são acionadas apenas quando você insere seus dados.
O Operador armazena Dados Pessoais: exclusivamente em meio eletrônico seguro; Sistema que utiliza Processamento Automatizado, exceto nos casos em que o Processamento não automatizado seja necessário em conexão com o cumprimento de requisitos legais; usando bancos de dados localizados no território da organização; a partir da data de recebimento do Consentimento e durante o período de sua validade.
O Operador não distribui ou fornece Dados Pessoais a terceiros sem Consentimento, exceto nos seguintes casos: se for necessário fornecer dados pessoais para atingir os objetivos de seu Processamento a terceiros, bem como se terceiros estiverem envolvidos em a prestação de serviços para tais fins (transportadoras, empresas de transporte); ao usar programas analíticos; para prevenir, suprimir ações ilegais do Titular dos Dados Pessoais, proteger os interesses legítimos do Operador e de terceiros; a pedido fundamentado do judiciário, das autoridades de segurança do Estado, do Ministério Público, da polícia, das autoridades investigadoras, de outros órgãos e organizações nos casos estabelecidos por atos normativos vinculantes.
O operador avisa os destinatários que os dados transmitidos só podem ser utilizados para os fins para os quais são comunicados.
Em caso de confirmação do fato de inexatidão dos Dados Pessoais, o Operador os atualiza.
O Operador destrói Dados Pessoais nos seguintes casos: existe uma ameaça à segurança dos Serviços; expiração do período de armazenamento de Dados Pessoais; a pedido do Utilizador.
O Operador exclui todas as informações recebidas do Usuário quando ele revoga o Consentimento. O direito de usar os Serviços é rescindido.
O Operador tem o direito de interromper temporariamente o Processamento (a menos que o Processamento seja necessário para esclarecer os Dados Pessoais).

Os Titulares dos Dados Pessoais são os Usuários que: forneceram dados nos formulários do Sistema.
